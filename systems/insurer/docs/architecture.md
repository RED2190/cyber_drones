# ЦПБ
### Цели безопасности
1.	Информация о заказах (маршрут, вес, содержимое, стоимость) доступна только строго определенному кругу лиц и сервисов (дрон, эксплуатант, страховая при наступлении страхового случая).
2.	Страховые выплаты и активация страховых полисов происходят только при наступлении достоверных страховых случаев.
### Предположения безопасности
1.	Аутентичные сервисы, с которыми интегрируется страховщик, благонадежны и безопасны.
2.	Сервисы страховщиков развернуты в доверенной среде, где гипервизор и ОС не скомпрометированы. (в рамках учебного проекта)
